mở rổ phim rồi lấy 1 file phim và 1 file sub (nếu extenssion hiện 1 file sub thì sub đã được dán trong phim nên không cần tải về)

bước 2: là sau khi lấy link phim thì chạy lệnh này trong linux để tải phim về sau đó kéo về máy cần xem
các ứng dụng cần biết để tải về là (ubuntu):


sudo apt update && sudo apt install -y yt-dlp aria2 ffmpeg
sudo apt install -y pipx
pipx ensurepath
pipx install --force yt-dlp


sau đó chạy lệnh này và tận hưởng thôi
yt-dlp --downloader aria2c --downloader-args "aria2c:-c -x 16 -s 16 -k 32M --summary-interval=1" --concurrent-fragments 64 --retries infinite --fragment-retries infinite --no-abort-on-unavailable-fragment "<link phim tải>" -o "<tên phim>.mp4"
